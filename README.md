Link: https://weather-today.vercel.app

## ✨ Tecnologias

- HTML5/CSS3
- Sass
- ReactJS

## 🚀 Executando localmente

- Clone o repositório
- Instale as dependências com `yarn`
- Execute com `yarn start`

Agora você pode acessar [`localhost:3000`](http://localhost:3000) do seu navegador.
